Uses MMH3 implementation from smhasher: https://github.com/aappleby/smhasher

Compile by running buildWindowsDLL.bat on Windows or buildLinuxSO.sh on Linux.

Requires g++
See here for how to set up mingw/g++ on Windows
https://code.visualstudio.com/docs/cpp/config-mingw

Linux version built on WSL Ubuntu, other distros should work though
